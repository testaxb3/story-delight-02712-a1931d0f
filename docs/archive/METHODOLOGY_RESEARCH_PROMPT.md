## PROMPT:

Preciso de dados científicos específicos e estatísticas para validar uma metodologia de parenting scripts baseada em neurociência. O produto é para pais de crianças com 3 perfis neurológicos específicos:

### 1. DEFIANT (Oppositional/High Autonomy Drive)
Busque:
- Dados sobre anterior cingulate gyrus hyperactivity em crianças oposicionistas
- Estudos sobre BAS (Behavioral Activation System) e BIS (Behavioral Inhibition System) em crianças
- Pesquisas de Ross Greene sobre Collaborative Problem Solving (CPS) - taxas de eficácia
- Redução de cortical thickness em crianças com ODD/oppositional behavior
- Alterações em dopamina/serotonina relacionadas a comportamento desafiador

### 2. INTENSE (Highly Sensitive/Emotionally Reactive)
Busque:
- Percentual de crianças com alta sensibilidade (HSC - Highly Sensitive Child)
- Pesquisas sobre limbic system reactivity em crianças sensíveis
- Dados sobre sensory processing disorder (SPD) prevalência
- Estudos sobre "dual susceptibility" em crianças HSC
- Research sobre co-regulation e mirror neurons em parent-child dyads

### 3. DISTRACTED (Executive Function Challenges/ADHD Traits)
Busque:
- Dados sobre prefrontal cortex development timeline (ages 2-8)
- Working memory limitations em crianças pequenas
- Estudos sobre hyperfocus em crianças com ADHD
- Pesquisas sobre external scaffolding para executive function deficits
- Neuroplasticity e myelination durante desenvolvimento infantil

### FORMATO IDEAL DA RESPOSTA:

Para cada tópico, forneça:
1. **Estatística específica** (ex: "15-20% das crianças são HSC")
2. **Fonte acadêmica** (autor, ano, publicação)
3. **Achado neurológico** (qual região cerebral, neurotransmissor, etc.)
4. **Implicação prática** (como isso afeta comportamento)

### EXEMPLO DO QUE PROCURO:

❌ RUIM: "Crianças com ADHD têm dificuldades executivas"

✅ BOM: "Crianças com ADHD mostram 10-13% de redução em cortical thickness no ventromedial prefrontal cortex (Shaw et al., 2007), região crítica para impulse control e decision-making, explicando por que comandos multi-step são neurologicamente difíceis de processar."
